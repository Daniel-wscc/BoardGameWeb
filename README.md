# BoardGameWeb

BoardGameWeb 是一款簡單的網頁遊戲，目前收錄一款經典陣營類桌遊，最大可同時10位玩家同時遊玩。

## DEMO網頁

- https://bgweb.online/

## 技術細節

- **前端：** 使用 JavaScript、HTML 和 CSS 開發。
- **後端：** 使用 Node.js、 Express、 WebSocket 技術。
